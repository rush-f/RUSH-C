## 💻 기술 스택 & 개발 환경

- Backend  : Spring Boot, Spring Data JPA, junit, Spring Security  
- Frontend : React, React-google-maps  
### 언어
- Backend : Java
- Frontend : HTML/CSS/JS
- Database : SQL

### 서버
- Raspberry Pi 3 Model B Plus Rev (OS : Raspbian GNU/Linux 10)
